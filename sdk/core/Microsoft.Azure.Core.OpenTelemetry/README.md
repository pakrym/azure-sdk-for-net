# Newtonsoft.Json support for Azure Core shared client library for .NET

The [Azure.Core package][azure_core_package] contains types shared by all Azure SDK client libraries.
This library contains OpenTelemetry collectors for all Azure.Core-based libraries.

## Getting started

Install this package if you want to collect Azure SDK tracing data using OpenTelemetry

### Install the package

Install this package from [NuGet] using the .NET CLI:

```bash
dotnet add package Microsoft.Azure.Core.OpenTelemetry
```
